# Sumário rápido — SimpleForNotes

Landing estática: hero, recursos, como funciona, preview, depoimentos, CTA final, nav e rodapé. Responsiva.

**Stack:** HTML, CSS (variáveis + tema escuro/claro), JS vanilla. Fonte Inter (Google Fonts). Sem framework.

**Onde está o quê:** conteúdo no `index.html`, visual no `css/styles.css`, comportamento no `js/script.js` (+ `config.js` se quiser centralizar constantes). Mídia em `assets/`, dados de exemplo em `data/content.json`. Docs extra em `docs/`.

**Ideias pra depois (se quiser):** imagens reais, formulário/newsletter, analytics, mais páginas (preços, blog), deploy (Netlify/Vercel/GitHub Pages — vê `docs/DEPLOYMENT.md`).
